# php_mysql_docker_ubuntu_2004

Docker image for PHP & MySQL. Built with Ubuntu 20.04 packages

### build
```
./docker build
```

### start
```
./docker up
```

### start (as daemon)
```
./docker upd
```

### exec
```
./docker ex web
```

### URL
http://localhost:8001/phpinfo.php
